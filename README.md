# Telegram Edit Any Message - iOS Tweak

Твик для Telegram, который позволяет локально редактировать любые сообщения (изменения видны только у вас).

[![Build Status](https://github.com/ofissed/EditTweak/workflows/Build%20iOS%20Tweak/badge.svg)](https://github.com/ofissed/EditTweak/actions)

## Возможности

- ✏️ Добавляет кнопку "Изменить" в контекстное меню любого сообщения
- 💾 Сохраняет изменения локально на устройстве
- 🔄 Изменения сохраняются после перезапуска приложения
- 🎨 Простой UI для редактирования текста

## Установка

### Автоматическая сборка через GitHub Actions

1. Форкни этот репозиторий или создай свой
2. Загрузи код на GitHub
3. GitHub автоматически скомпилирует твик
4. Скачай .deb файл из раздела **Actions** → выбери последний build → **Artifacts**
5. Перенеси .deb на iPhone
6. Установи через Filza
7. Respring

### Ручная установка на iPhone

Если у тебя установлен Theos на iPhone:

```bash
cd /путь/к/проекту
make package
make install
killall -9 SpringBoard
```

## Как использовать

1. Открой Telegram
2. Долгое нажатие на любое сообщение
3. Нажми "✏️ Изменить"
4. Введи новый текст
5. Нажми "Сохранить"

**Важно:** Изменения видны только на твоем устройстве. На сервере сообщения не меняются.

## Требования

- iOS 14.0+
- Джейлбрейк (Dopamine, Palera1n, Checkra1n и т.д.)
- Telegram установлен

## Разработка


### Локальная компиляция

Требуется Theos. Смотри `SETUP_WINDOWS.md` или `SETUP_NO_WSL.md` для инструкций.

```bash
make clean
make package
```

## Известные проблемы

- Код использует базовые классы Telegram, которые могут измениться в новых версиях
- Требуется адаптация под конкретную версию Telegram
- Для полной функциональности нужно найти правильные классы через Flex 3
